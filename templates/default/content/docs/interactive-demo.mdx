This page demonstrates **MDX support** - combining Markdown with Svelte components.

## Live Components

Below are real Svelte components rendered inside markdown:

## Regular Markdown Still Works

- Lists work normally
- **Bold** and *italic* text
- [Links](/docs) work too

```javascript
// Code blocks also work
const greeting = "Hello from MDX!";
console.log(greeting);
```

## Why MDX?

1. Write content in familiar Markdown
2. Embed interactive Svelte components
3. No need for custom shortcodes
4. Full component props support
